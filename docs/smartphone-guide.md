# スマホ操作手順

## 必要なもの

- Tailscaleアプリ（インストール・ログイン済み）
- Safari（iOS）または Chrome（Android）

---

## 毎回の接続手順

1. **PC側でサーバーを起動**してもらう（または自分で起動する）
2. **Tailscaleアプリを開いて**接続状態を確認する
3. ブラウザで以下にアクセスする：

```
http://100.65.1.43:8000
```

---

## 画面の見方

```
┌─────────────────────────────┐
│ ● Voice Dev  [パス] [リセット]│  ← ヘッダー
├─────────────────────────────┤
│                             │
│  [AIの返答]                  │
│                    [自分の発言]│
│  ⚙ run_shell                │  ← ツール実行ログ
│  git commit...              │
│                             │
├─────────────────────────────┤
│  🎤  タップして話す           │  ← マイクボタン
│ ┌───────────────────┐  [↑] │
│ │ テキスト入力...    │      │  ← テキスト入力
│ └───────────────────┘      │
└─────────────────────────────┘
```

- **左上の●**: 緑 = サーバー接続中 / 赤 = 未接続
- **パス表示**: 作業中のプロジェクトフォルダ（タップで変更）
- **リセット**: 会話履歴をクリア

---

## 音声入力の使い方

1. **「タップして話す」ボタン**を押す
2. ボタンが赤くなったら話す
3. 話し終えると自動でAIに送信される

> iOSはSafari、AndroidはChromeを使うと音声認識が安定する。

---

## テキスト入力の使い方

テキストエリアに直接入力して **↑ボタン** または **Enterキー** で送信。  
Shift+Enter で改行できる。

---

## プロジェクトを切り替える

ヘッダーのパス表示（例: `~/`）をタップするとダイアログが出る。  
開発したいプロジェクトのフルパスを入力する。

例：
```
/home/kokinagano/myapp
```

設定はブラウザに保存されるので次回アクセス時も引き継がれる。

---

## AIへの話しかけ方の例

### プロジェクト確認
```
「このプロジェクトのファイル構成を教えて」
「package.jsonの内容を見せて」
```

### コーディング
```
「ログイン画面をHTMLで作って」
「この関数にエラーハンドリングを追加して」
「TypeScriptに書き直して」
```

### git操作
```
「変更をコミットして」
「featureブランチを作って」
「mainにマージして」
```

### 実行・確認
```
「npm installしてサーバーを起動して」
「テストを実行して結果を教えて」
「エラーが出てるので直して」
```

---

## 会話をリセットする

右上の **「リセット」** ボタンを押すと会話履歴がクリアされる。  
プロジェクトを切り替えるときやAIが混乱しているときに使う。

---

## 使い終わったら

1. ブラウザを閉じる（またはタブを閉じる）
2. **Tailscaleアプリを開いてOFFにする**（バッテリー節約のため）

PC側のサーバーを止めてもらうことも忘れずに。

---

## つながらないときの確認手順

1. TailscaleアプリがONになっているか確認する
2. ブラウザのURLが `http://100.65.1.43:8000` になっているか確認する
3. PC側でサーバーが起動しているかPCの人に確認する
4. Tailscaleアプリを一度OFFにしてONに戻してみる
