# UI全面刷新 — デザインシステム導入

> ステータス: **実装済み**
> 作成日: 2026-07-06 / 実装完了: 2026-07-06

## 目的

機能は一切変えずに、`client/index.html` の画面設計・UIを大幅に向上させる。
「音声ファーストの開発コンパニオン」として、スマホでも PC でも美しく使える洗練されたダークUIにする。

## 制約（機能を変えないための原則）

- **JSが参照する id / class 名は一切変更しない**（`mic-btn`, `status-dot`, `mode-btn.active`, `fp-*`, `plan-*`, `idea-*`, `proposal-*`, `autodev-*` など）
- HTML構造の変更は「装飾専用要素の追加」のみ（ブランドマーク、シートのグラブハンドル等）
- JS変更は「インラインstyleをclassに置き換える」等の見た目に限る

## デザインシステム

### カラートークン

| トークン | 値 | 用途 |
|---------|-----|------|
| `--bg` | `#05070c` + 微細なラジアルグラデ | 最深層背景 |
| `--surface` / `--surface-2` | `#0f1522` / `#151d2e` | カード・パネル階層 |
| `--border` / `--border-strong` | `#1f2937` / `#2e3b52` | 境界の強弱 |
| `--text` / `--muted` / `--muted-2` | `#e3e9f3` / `#93a0b4` / `#5c6b82` | 文字階層 |
| `--accent` | `#5b9dff` | 主要アクション |
| グラデーション | `#4c8dff → #7c5cff` | 送信・ユーザーバブル・ブランド |
| モード色 | 定義 `#b98aff` / 実装 `#5b9dff` / 企画 `#e3b341` / 提案 `#ff9254` | モード識別 |

### 主要コンポーネントの刷新

1. **ヘッダー**: 半透明+ブラー(ガラス)、ブランドマーク🎙（接続状態ドットをマーク右下に統合）、モードトグルはセグメントコントロール化（アクティブ時にモード色のティント+グロー）
2. **メッセージ**: ユーザー=グラデーションバブル、AI=ソフトカード。出現アニメーション（fade+slide）。Markdownのテーブル・引用・コードを整備
3. **ツール実行カード**: ターミナル風（macOS風3ドット+コマンド名、等幅フォント）
4. **思考中インジケータ**: グラデーションオーブ+波打つドット+経過タイマー
5. **マイクボタン（ヒーロー要素）**: 大型ピル、アイドル時は呼吸グロー、リスニング時は赤グラデ+波紋リング拡散アニメーション
6. **入力エリア**: ブラー背景、フォーカスリング、グラデーション送信ボタン、safe-area対応
7. **ピッカー**: ワークスペース=大型タイル（ホバーで浮上+モード色リング）、プロジェクト=行カード+シェブロン
8. **アイデアカード**: ステータス別の左アクセントボーダー、★はポップアニメーション+ゴールドグロー
9. **提案リスト**: カスタムチェックボックス、角度バッジチップ、完了は打ち消し+減光
10. **自動開発バー**: アンバーグラデパネル、ON時はロボットグロー演出
11. **モーダル**: ボトムシートにグラブハンドル、スプリング風スライドアップ、深いブラー
12. **トースト/選択肢/チップ**: ガラス化・グラデ枠・角丸統一
13. **スクロールバー**: スリムカスタム
14. **レスポンシブ**: デスクトップでは本文を最大 ~840px に中央寄せ、モバイルは `env(safe-area-inset-*)` 対応
15. **アクセシビリティ**: `:focus-visible` リング、`prefers-reduced-motion` 対応

### 実装メモ

- JSが参照する id / class は全て維持。HTML変更はブランドマーク・シートハンドル追加と `impl-start-bar` のclass化のみ
- ヘッドレス検証時の豆腐文字は検証環境に日本語/絵文字フォントが無いためで、実機では正常
- 検証用に `server/.venv` に playwright を追加インストール（requirements.txt には含めない開発用）
